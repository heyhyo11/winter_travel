# winter travel
**프로젝트 소개**   
Winter Travel Project는 겨울이라는 계절에 맞는 여행지를 사용자에게 추천해주는 서비스입니다. 메인 피드에는 3가지 방식으로 각 사용자마다 다른 추천 여행지가 등장합니다. 첫 번째는 머신러닝을 이용한 유사 사용자의 관심사, 두 번째는 일치하는 카테고리순, 세 번째는 랜덤으로 추천됩니다.
  
**개발기간**  
2022.01.26 ~ 02.09
